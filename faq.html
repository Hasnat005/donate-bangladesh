Important FAQ

What is the Document Object Model (DOM)?

The DOM is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree of objects, where each node corresponds to an element or attribute. JavaScript can interact with the DOM to dynamically update content, style, and structure.

How do you select an element from the DOM?

You can select elements from the DOM using various JavaScript methods such as:

  • document.getElementById('id'): Selects an element by its ID.
  • document.querySelector('selector'): Selects the first element matching a CSS selector.
  • document.querySelectorAll('selector'): Selects all elements matching a CSS selector.
  • document.getElementsByClassName('class'): Selects elements by their class name.
What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where a single event listener is added to a parent element to manage events for all of its child elements. This is useful for handling dynamic content and improves performance, especially when managing many elements or when new elements are added dynamically.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate attributes and styles using JavaScript methods like:

  • element.setAttribute('attribute', 'value'): Modifies or adds an attribute.
  • element.removeAttribute('attribute'): Removes an attribute.
  • element.style.property = 'value': Modifies the CSS styles of an element (e.g., element.style.backgroundColor = 'blue').
  • element.classList.add('class'), element.classList.remove('class'): Adds or removes CSS classes for styling.
